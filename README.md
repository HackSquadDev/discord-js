## Discord-JS

This is a simple discord bot made with discord.js for HackSquad's discord server.

### How to use

1. Clone the repo
2. Run `npm install`
3. Create a file called `.env` and add the following line to it: `TOKEN=YOUR_TOKEN_HERE`
4. Run `npm start`

### How to contribute

1. Fork the repo
2. Make your changes
3. Create a pull request
